# 四季穿搭灵感网站

一个优雅的穿搭展示网站，使用文学化的季节表达方式，让用户可以按季节浏览不同的穿搭风格。

## 功能特点

### 🌸 文学化季节表达
- **春暖花开** - 春季穿搭，轻盈优雅
- **夏日炎炎** - 夏季穿搭，清爽舒适  
- **秋高气爽** - 秋季穿搭，温暖层次
- **冬日暖阳** - 冬季穿搭，温暖时尚

### ✨ 用户体验
- **响应式设计** - 完美适配桌面、平板和手机
- **平滑切换** - 流畅的季节切换动画
- **图片懒加载** - 优化加载性能
- **触摸滑动** - 移动端支持左右滑动切换
- **键盘导航** - 支持左右方向键切换
- **滚动到顶部** - 智能显示返回顶部按钮

### 🎨 视觉效果
- **渐变背景** - 优雅的头部渐变设计
- **悬停效果** - 卡片悬停时的缩放和阴影效果
- **主题色彩** - 每个季节独特的主题色彩
- **现代布局** - 网格布局展示穿搭卡片

## 使用方法

1. **直接打开** - 双击 `index.html` 文件即可在浏览器中查看
2. **季节切换** - 点击顶部导航栏的季节标签
3. **浏览穿搭** - 滚动页面查看不同季节的穿搭图片
4. **交互操作**：
   - 点击穿搭卡片查看详情
   - 使用左右方向键切换季节
   - 在移动端左右滑动切换季节

## 技术特性

- **纯前端实现** - 无需后端服务
- **现代CSS** - Grid布局、Flexbox、CSS动画
- **原生JavaScript** - 无需外部依赖
- **性能优化** - 图片懒加载、防抖节流处理
- **错误处理** - 图片加载失败时的优雅降级

## 自定义建议

您可以通过以下方式自定义网站：

1. **更换图片** - 修改HTML文件中的图片URL
2. **调整颜色** - 修改CSS中的主题色彩变量
3. **添加内容** - 在每个季节区域添加更多穿搭卡片
4. **修改文案** - 更新季节标题和穿搭描述
